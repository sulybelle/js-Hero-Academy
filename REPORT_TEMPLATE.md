# Лабораториялық жұмыс №4 - Отчет шаблоны

## Тақырып
React жобасын құру және құрылымды теңшеу

## Мақсаты
- React ортасын орнату
- Жоба құрылымын ұйымдастыру
- Компоненттік архитектураны қолдану

## 1. React жобасын құру (Сурет 1.1)
Бұл бөлімде `npm run dev` нәтижесін және браузерде ашылған бетті салыңыз.

## 2. Жоба құрылымы (Сурет 1.2)
`src/components`, `src/pages`, `src/assets`, `src/styles` құрылымын скриншотпен көрсетіңіз.

## 3. Компоненттер (Сурет 1.3)
`Header`, `Footer`, `Card`, `Button`, `Form` код фрагменттерін көрсетіңіз.

## 4. JSX, Props, map, useState (Сурет 1.4)
`HomePage.jsx` және `Card.jsx` ішінен дәлел болатын кодтарды қосыңыз.

## 5. Алдыңғы жобадан көшіру (Сурет 1.5)
Search және Form validation жұмысын скриншотпен көрсетіңіз.

## Қорытынды
Жоба React-қа сәтті көшірілді және ТЗ-дағы барлық талап орындалды.
